SubWire¶

class pyhdl.SubWire(node, sub, type='undefined')¶

A slice of a wire.

You can index into a wire using standard array notation to create a slice. e.g:

>>> from pyhdl import ConstantWire
>>> a = ConstantWire(value="010101", width=6)
>>> b = a[0]
>>> b.val
"0"
>>> c = a[0:2]
>>> c.val
"01"
Parameters:
  • node – The wire you want to slice.
  • sub – The slice you want to take.
  • type (str) – An arbitrary string to tag a wire. (DEPRECATED)

Simple combinatorial gates¶

Named parameters from a to z are used as inputs, and out is used as the output. (Except in the case of the not gate, where inp is used as the input.)

width is used to specify the width of the inputs, and ways is used to specify the arity of the gate.

For example::

from pyhdl import NandGate, Wire
a, b, c, out = Wire(), Wire(), Wire(), Wire()
gate = NandGate(a=a, b=b, c=c, out=out, ways=3)

Multiplexer¶

class pyhdl.Multiplexer(width=1, ways=2, **kwargs)¶

A Multiplexer.

Named parameters from ‘a’ to ‘z’ are used as inputs. e.g.:

from pyhdl import *
a, b, c, d = Wire(), Wire(), Wire(), Wire()
sel = Wire(width=2)
out = Wire()
nand = Multiplexer(a=a, b=b, c=c, d=d, out=out, sel=sel, ways=4)
Parameters:
  • out – The output wire.
  • sel – The selector wire.
  • width (int) – The width of the inputs.
  • ways (int) – The number of inputs to the multiplexer.

Demultiplexer¶

class pyhdl.Demultiplexer(width=1, ways=2, **kwargs)¶

A Demultiplexer.

Named parameters from ‘a’ to ‘z’ are used as outputs. e.g.:

from pyhdl import *
a, b, c, d = Wire(), Wire(), Wire(), Wire()
sel = Wire(width=2)
input = Wire()
nand = Multiplexer(a=a, b=b, c=c, d=d, input=input, sel=sel, ways=4)
Parameters:
  • input – The input wire.
  • sel – The selector wire.
  • width (int) – The width of the outputs.
  • ways (int) – The number of outputs from the demultiplexer.
